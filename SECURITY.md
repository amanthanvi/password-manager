# Security Policy

## Reporting vulnerabilities
Please report security issues privately to `amanthanvi+npw-security@gmail.com`.
Do not open public issues for exploitable vulnerabilities.

Include:
- affected version/commit
- reproduction steps
- impact assessment
- any proof-of-concept

## Supported versions
- `main` branch (active development)
- Latest tagged release line (`v0.1.x` once released)

## Response targets
- Initial acknowledgement: within 7 days
- Triage + severity assignment: within 14 days
- Remediation target: within 90 days (critical issues are prioritized faster)

## Disclosure and credit
Coordinated disclosure is preferred. We credit reporters in release notes unless anonymity is requested.
